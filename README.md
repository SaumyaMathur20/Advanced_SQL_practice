# 📊 Advanced SQL Practice 

This repository contains my hands-on SQL practice focused on strengthening problem-solving and query-writing skills.

Instead of limiting practice to only basic queries, I worked through structured SQL questions covering subqueries, CTEs, window functions, stored procedures, CASE statements, date and string functions, conditional aggregation, views, indexes, and other important MySQL concepts.

The goal of this repository is to demonstrate practical understanding of intermediate to advanced SQL concepts through a self-solved question-and-answer format.

---

## 🎯 Project Objective

The purpose of this practice repository was to improve my ability to:

- write clean and structured SQL queries
- identify the right SQL approach based on the problem statement
- revise advanced SQL concepts through hands-on practice

---

## 🗂 Dataset Used

A sample **Hospital Management** dataset was used for practice. The dataset includes the following tables:

- **patients** – patient details such as patient ID, name, city, and admission date
- **appointments** – appointment-level data including patient ID, doctor ID, appointment date, and consultation fee
- **billing** – payment details for appointments, including payment mode and payment status

This dataset was used to simulate realistic SQ questions in a structured way.
